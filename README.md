# @benparnell/tsconfig

A basic tsconfig I sometimes remember to use on my TypeScript projects.

## Install

```sh
npm install --save-dev @benparnell/tsconfig
```

## Usage

Add this to your tsconfig.json:

```json
{
  "extends": "@benparnell/tsconfig",
  "compilerOptions": {
    // Your options go here
  }
}
```
